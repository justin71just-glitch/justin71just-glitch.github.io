# Justin Han Personal Website

A personal introduction website built to present my identity, interests, learning direction, AI-assisted creation practice, and portfolio-style personal brand.

The website combines scroll-based visual storytelling, animated cards, image sections, and AI-assisted content creation to introduce who I am: a finance student, an AI enthusiast, a frontier innovation explorer, and someone who values beauty, inclusiveness, freedom, and continuous self-growth.

## Overview

This website is designed as a personal homepage rather than a traditional static resume page.

It uses a visual, animated, scroll-driven structure to introduce:

- personal identity and background;
- finance learning and academic interests;
- AI and frontier technology exploration;
- aesthetics, photography, fitness, and lifestyle interests;
- data analysis, report writing, product design, and AI creation practices.

The homepage opens with a hero section titled **“Justin and Han”** and introduces the site as **“han's personal website”**. It presents a welcoming message, animated chat bubbles, a progress indicator, and a visual character element to create a playful first impression.

## Main Sections

The hero section introduces the site with:

- the phrase **“WELCOME TO MY WEBSITE”**;
- a short self-description emphasizing innovation, beauty, inclusiveness, endless possibilities, and openness to life experiences;
- a progress-style visual element.

The website includes several sticky cards introducing different sides of myself:

#### Innovation

Shows my interest in frontier technologies, theoretical innovation, artificial intelligence, AI agents, and how emerging technologies reshape finance, economic systems, and the future.

#### Explorer

Introduces my broader personal interests, including film, sound, painting, photography, composition, running, fitness, cinema, music, self-fulfillment, deep communication, and human connection.

## AI-Assisted Building Process

This site was created with the help of AI tools, including:

- Claude Code
- Kimi
- DeepSeek V4 Pro
- ChatGPT

AI tools were used to support:

- website structure planning;
- page content organization;
- copywriting refinement;
- visual expression exploration;
- code generation and modification assistance;
- animation and effect reproduction;
- section layout iteration;
- image generation and creative direction.

The project is not only a personal website, but also a practical experiment in AI-assisted website building and AI-native personal workflow design.
